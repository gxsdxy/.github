## 📖 About

Hi! I'm a computer science teacher at Guangxi Vocational College of Water Resources and Electric Power. This is where I keep the courseware, sample code, and exercises used in my classes — students and fellow teachers are welcome to use them.

## 📂 What's Inside

| Repo Type | Contents |
| --- | --- |
| 📑 Courseware | Lecture notes and slides for each course |
| 💾 Sample Code | Example programs used in class demos |
| ✏️ Exercises | Practice problems and reference solutions |
| 🧰 Tools & Resources | Setup guides and learning material links |

## 🎓 For Students

- Each course has its own repository — find it by course name.
- If you have questions, open an Issue in the relevant repository.
- Feel free to star ⭐ and fork, but please complete your own assignments independently.
